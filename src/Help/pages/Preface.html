Labstache

April 2020

Thank you for using the Labstache LabVIEW toolkit. Labstache provides a API to render logic-free Mustache templates. No external dependencies are required. The template compiler is 100% LabVIEW and can be used anywhere the LabVIEW run-time engine is installed.

Installation

This section documents installation instructions for the Labstache LabVIEW toolkit.

  1. Download the vipm package for Labstache.
  2. Open it with VI Package Manager (available for free at www.jki.net/vipm) and follow the on-screen instructions.
  3. Restart LabVIEW.
