Array Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Array class represents an array of Mustache data.

Class Item Description
From Wraps, or converts, an array of LabVIEW values into an Mustache array.
Read Type Gets the Mustache data type of an object. This is primarily used internally or when building custom compilers. This is more robust than using strings in case structures with the Get LV Class Name function, which is not available in older versions of LabVIEW.
Read Value Gets the internal array of Mustache data.
