Data Classes, Controls, and VIs

Owning Palette: Labstache Classes, Controls, and VIs

Requires: Labstache

The Data project library contains all the Mustache-related data types. Each data type is represented as a LabVIEW Class, which can be used to convert, or wrap, a LabVIEW value into a Mustache data type and value. The data type is a desendant of the Datum LabVIEW class within this project library. The Datum acts as a base class for all data types and should not be used directly. Instead, a sub-class should be created from the Datum class.

Palette Object Description
Array The Array class represents an array of Mustache data.
Boolean The Boolean class represents a Mustache boolean (true or false) data type and value.
Datum The Datum class is the abstract base class for all data types that are recognized by the compiler and used to control the behavior of the rendering process. These are the data types that are recognized by a Mustache template and are largely identical to the JavaScript/JSON data types and primitatives. Objects of this class should not be directly created. Instead, a new class that inherints from the Datum class should be created.
Function The Function class represents a Mustache Lambda, or anonymous function, which is a callable object. This class is essentially an abstract class and should only be sub-classed with a concrete implementation. A Mustache Lambda receives the unrendered, raw input and outputs rendered text. This is useful for implementing filters or caching.
Map The Map class represents a collection of key-value pairs of Mustache data types and values.
Null The Null class represents a nothing, none, or NULL Mustache data type and value. A Null object should always be considered FALSE in truthiness and equality testing of Mustache data types and values.
Path The Path class represents a file or directory path within the Operating Sytem's file system. Strictly speaking, there is not an equivalent Mustache data type or value, but this is useful for implementing Partial Loaders based on file paths using the Loader class.
String The String class represents a Mustache String data type and value.
