Path Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Path class represents a file or directory path within the Operating Sytem's file system. Strictly speaking, there is not an equivalent Mustache data type or value, but this is useful for implementing Partial Loaders based on file paths using the Loader class.

Class Item Description
From Wraps, or converts, a LabVIEW path into a Mustache value.
Read Type Gets the Mustache data type of an object. This is primarily used internally or when building custom compilers. This is more robust than using strings in case structures with the Get LV Class Name function, which is not available in older versions of LabVIEW.
Read Value Gets the internal LabVIEW path value.
