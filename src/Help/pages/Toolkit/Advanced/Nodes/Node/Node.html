Node Controls and VIs

Owning Palette: Nodes Classes, Controls, and VIs

Requires: Labstache

The Node class is the abstract base class for all nodes in an Abstract Syntax Tree (AST). Objects of this class should not be directly created. Instead, a new class that inherents from the Node class should be created.

Class Item Description
Read Delimiters Gets the opening and closing components that surround, or delimit, a Mustache tag.
Read Source Gets the original, raw text from the Mustache template that yielded the node during the parsing process.
Read Type Gets the Node type of an object. This is primarily used internally or when building custom compilers. This is more robust than using strings in case structures with the Get LV Class Name function, which is not available in older versions of LabVIEW.
Write Delimiters Sets the opening and closing components that surround, or delimit, a Mustache tag.
