Text Controls and VIs

Owning Palette: Nodes Classes, Controls, and VIs

Requires: Labstache

The Text class represents a block of text without any Mustache tags as a node in an Abstract Syntax Tree (AST).

Class Item Description
New Creates a new Text node that represents a block of text without any Mustache tags.
Read Source Gets the original, raw text from the Mustache template that yielded the node during the parsing process.
