Labstache Classes, Controls, and VIs

The classes, controls, and VIs that are the Application Programming Interface (API) for the Labstrache toolkit.

Palette Object Description
Advanced
Compile Compiles a Mustache template using the types of data passed into the compiler to control the "how" and "what" is provided in the output. The type fo data for each tag and key can determine the rendering behavior of the node in the syntax tree, so LabVIEW classes are provided to represent the different types of data.
Data The Data project library contains all the Mustache-related data types. Each data type is represented as a LabVIEW Class, which can be used to convert, or wrap, a LabVIEW value into a Mustache data type and value. The data type is a desendant of the Datum LabVIEW class within this project library. The Datum acts as a base class for all data types and should not be used directly. Instead, a sub-class should be created from the Datum class.
